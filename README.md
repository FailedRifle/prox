# CheddarOS Proxy

A simple web proxy. Browse anything.

## Requirements

- [Node.js](https://nodejs.org/) — install this first

## How to run

**Windows** — double-click `start.bat`

**Mac / Linux**
```bash
chmod +x start.sh
./start.sh
```

**Manual**
```bash
npm install
npm start
```

Then open [http://localhost:8080](http://localhost:8080)
